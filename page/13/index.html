undefined 小结

undefined 类型的意思是当前对象未定义,适用于下面几种情况

  1. 变量声明,但未赋值
  2. 对象没有赋值的属性,该属性的值为undefined
  3. 调用函数参数,但是未提供参数，该参数的值为undefined
  4. 函数没有返回值时,默认返回undefined

  • name
  • class name

  • css selector
